# VNDK Compat Models

This directory stores API model JSON files used by the VNDK compatibility
pipeline (`vndk_compat.mk` → `vndk_diff_engine.py`).

## Generating a Model

Models are generated by scanning the system libraries of a built AOSP tree:

```bash
python3 build/make/tools/vndk_compat/vndk_api_model.py \
    --api-level 16 \
    --scan-dir out/target/product/<TARGET>/system/lib64 \
    --output build/make/tools/vndk_compat/models/v16.model.json
```

Replace `<TARGET>` with your lunch target directory (e.g., `treble_arm64_bvN`).

## File Naming

Models follow the convention `v<API_LEVEL>.model.json`:

| File | Source |
|------|--------|
| `v16.model.json` | Android 16 system libraries |

## When to Regenerate

Regenerate after any AOSP source sync that updates system libraries, or
when targeting a different Android version.
